# vuedemo

> This is a vue2.0 demo

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## 准备工作
``` bash
安装nodejs、npm、webpack
```
